# Python 与 Opencv 配置安装

  • 熟悉 Python、能用、能看懂 即可
  • 环境配置：
    • Python
    • Jupyter notebook
    • Opencv-Python
    • IDE（集成开发工具）

# Python

Anaconda 相当于Python全家桶，Python 和 用到工具包都有了。

  • Python 2 基本已经不用了，在官网选择Python 3版本

# Opencv-Python

  • 在 opencv 3.4.2 以后有些算法被申请了专利（比如教科书中关于特征提取的算法），这个开源实现就没办法用了
  • 我们可以使用 opencv 3.4.1.15 继续使用那些算法。

安装最新版本opencv

pip install opencv-python # 安装最新版本opencv

安装指定版本opencv

pip install opencv-python==3.4.1.15 # 安装指定版本opencv

安装测试

  • 在python交互式编程命令行执行
import cv2  			#导入 opencv库，如果接下来没有报错，证明安装成功了
cv2.__version__		#查看版本号
# 输出：'4.6.0'

opencv-contrib-python

opencv扩展包，版本号必须和opencv一样，这个包相当给opencv于加了一些额外的扩展，比如一些特征提取的算法

pip install opencv-contrib-python

Jupyter notebook

Jupyter Notebook是基于网页的用于交互计算的应用程序。. 其可被应用于全过程计算：开发、文档编写、运行代码和展示结果。

  • 安装 Anaconda 已经包括这个包，无需单独下载

  • 但在一些 Anaconda 还未支持的平台比如，Apple M1 使用的是 miniforge（剪裁版Anaconda不同实现），需要单独下载 Jupyter notebook ，在终端依次输入：

    conda install jupyter
pip install jupyter_contrib_nbextensions
jupyter contrib nbextension install --user
pip install jupyter_nbextensions_configurator
jupyter nbextensions_configurator enable --user
jupyter notebook	#运行

  • 在终端输入：jupyter notebook 运行 Jupyter notebook

  • 可以写代码、写文档
